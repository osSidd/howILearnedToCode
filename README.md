# Markdown cheatsheet

## Headings
# H1 ---> \#
## H2 ---> \##
### H3 ---> \###
#### H4 ---> \####
##### H5 ---> \#####
###### H6 ---> \######
<br/><br/>

## Strong 
**Strong** ---> \**word\** <br/>
__strong__ ---> \__word\__
<br/><br/>

## Emphasis
*Emphasis* ---> \*word\* <br/>
_Emphasis_ ---> \_word\_ 
<br/><br/>

## Strike through
~~Strike throught~~ ---> \~~word\~~
<br/><br/>

## Horizontal rule
---
---> \---
___
---> \___
<br/><br/>

## Escape characters
use \character
<br/><br/>

## Blockquotes
> Blockquotes ---> \>
<br/><br/>
